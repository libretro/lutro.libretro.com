love.input

Provides an interface to handle inputs. This is specific to lutro and doesn't exist in LÖVE.

This module is deprecated. Please use love.joystick instead.

love.input.joypad (deprecated)

Returns whether a joypad button is pressed.

pressed = love.input.joypad(port, button)

Where port starts at 1, and button is a string with one of these values:
